# ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ВЕРСИЯ 2.0) — ORIGINS: CHAOS ADDON (NEOFORGE 1.21.1)

## 1. ОБЩИЕ ПОЛОЖЕНИЯ

- **Название проекта:** Origins: Chaos Addon  
- **Платформа:** Minecraft 1.21.1, NeoForge  
- **Тип:** Аддон для мода Origins (NeoForge)  
- **Язык реализации:** Java 21, Gradle  
- **Объём:** 18 уникальных рас, каждая с **3 пассивными** и **3 активными** способностями (всего >100 JSON-файлов)

**Ключевое требование:** все расы должны **радикально менять геймплей** выживания и PvP, делая игру неузнаваемой. Активные способности должны иметь выраженные визуальные и звуковые эффекты (только ванильные или из библиотек), а также чёткие числовые балансы, вынесенные в конфигурацию.

---

## 2. ТРЕБОВАНИЯ К РАЗРАБОТЧИКУ

**Перед началом разработки разработчик ОБЯЗАН:**

- Изучить **официальную документацию** мода **Origins (NeoForge)** — разделы: JSON Data Formats, Types Reference, Guides по датапакам.
- Изучить документацию **Pehkui**, **Additional Entity Attributes**, **Apugli** (если доступен для NeoForge), **Bookshelf**, **Cardinal Components API**, **Cloth Config API**, а также **Origins Extra Keybinds**.
- Ознакомиться с **Auto Converter** для конвертации датапаков с Fabric на NeoForge (если потребуется).
- Понимать, что NeoForge-версия Origins имеет **отличия в форматах** от Fabric-версии — все JSON должны писаться **с нуля** под NeoForge.

## 4. АРХИТЕКТУРА

- **Data-Driven:** все расы и способности определяются в JSON-файлах датапака.
- **Путь датапака:** мод создаёт папку `chaos_addon` с `pack.mcmeta` и структурой `data/chaos_addon/origins/`, `data/chaos_addon/powers/`, `data/chaos_addon/origin_layers/`.
- **Автозагрузка:** датапак помещается в папку `.minecraft/config/originpacks/` и автоматически подхватывается Origins.
- **Конфигурация:** через Cloth Config API создаётся экран, где игрок может менять все числовые параметры (кулдауны, урон, шансы, длительности и т.д.) для каждой расы.
- **Компоненты:** Cardinal Components API используется для хранения данных, специфичных для рас (см. ниже).

## 5. ОБЩИЕ ПРАВИЛА ДЛЯ СПОСОБНОСТЕЙ

### 5.1. Пассивные способности
- Действуют постоянно или при выполнении условия.
- Типы: `attribute_modifier`, `effect`, `conditioned_attribute`, `disable_regen`, `prevent_item_use` и т.д.

### 5.2. Активные способности
- Активируются нажатием клавиш (G, H, J — для primary, secondary, ternary).
- Имеют **кулдаун** (в секундах) и, возможно, **стоимость** (здоровье, голод, опыт, ресурсы).
- Отображаются в HUD (иконка и полоса кулдауна).

**Клавиши (с учётом Extra Keybinds):**
- `key.origins.primary_active` — 1-я активка (G)
- `key.origins.secondary_active` — 2-я активка (H)
- `key.origins.ternary_active` — 3-я активка (J)

### 5.3. Визуальные и звуковые эффекты
- Для каждой активной способности должны быть **чётко прописаны** используемые частицы и звуки.
- Используются **только ванильные звуки** и **частицы** (перечень прилагается ниже).
- При возможности — применять эффекты из Pehkui (изменение размера, вращение) и из дополнительных атрибутов.

## 6. ДЕТАЛЬНОЕ ОПИСАНИЕ ВСЕХ 18 РАС

Ниже для каждой расы перечислены:
- **Название**, **иконка** (предмет для отображения в меню выбора)
- **Пассивные способности** (3 штуки) — краткое описание и числовые параметры
- **Активные способности** (3 штуки) — подробно: название, действие, числовые параметры, **визуальные эффекты** (какие частицы, где появляются, как движутся) и **звуковые эффекты** (какой звук и когда проигрывается)

Все числовые параметры (кулдауны, урон, длительности, шансы) должны быть вынесены в Cloth Config и иметь значения по умолчанию, указанные ниже.

---

### 6.1. Пожиратель Миров (Eater of Worlds)
**Иконка:** `minecraft:nether_star`

**Пассивные:**
1. **Хаотическая аура** — каждые 30 секунд в радиусе 10 блоков все блоки имеют 20% шанс превратиться в случайный блок (земля→гравий→песок→стекло и т.д.).  
2. **Голод бесконечности** — игрок не может есть обычную еду; голод восполняется только поглощением опыта (1 уровень = +3 очка сытости). Без опыта голод падает в 2 раза быстрее.  
3. **Безумный шёпот** — все мобы в радиусе 15 блоков имеют 10% шанс каждые 5 секунд атаковать ближайшую цель (включая других мобов), игнорируя игрока.

**Активные:**
1. **Сдвиг реальности** (кулдаун 120 сек) — меняет местами два любых блока в радиусе 20 блоков (даже bedrock, но только на 10 секунд, затем возвращается).  
   - *Эффект:* вокруг обоих блоков появляются частицы `minecraft:portal` в течение 2 секунд.  
   - *Звук:* `block.portal.trigger` в момент обмена.  
2. **Призыв из Бездны** (кулдаун 120 сек) — в случайной точке в радиусе 50 блоков появляется босс (аналог Warden, 200❤, атакует всех, живёт 60 секунд).  
   - *Эффект:* вспышка чёрного дыма (`minecraft:smoke`) и частицы `minecraft:dragon_breath` в месте спавна.  
   - *Звук:* `entity.warden.sonic_boom` и `entity.ender_dragon.death`.  
3. **Временная петля** (кулдаун 180 сек) — создаёт зону радиусом 15 блоков, в которой все действия (движение, атаки, использование предметов) повторяются через 2 секунды с задержкой (эффект «эхо»). Длится 10 секунд.  
   - *Эффект:* внутри зоны постоянно летают частицы `minecraft:sculk_charge` и `minecraft:sculk_soul` с зеленоватым оттенком.  
   - *Звук:* циклическое проигрывание `entity.elder_guardian.curse` и `block.portal.ambient`.

**Цена за активки:** каждая активка отнимает 5 уровней опыта. Если опыта меньше — способность не срабатывает.

**Недостатки:** кровать взрывается (сила TNT) при попытке сна; солнечный свет наносит 0.5❤ урона каждые 5 секунд (как у зомби).

### 6.2. Повелитель Роя (Swarm Lord)
**Иконка:** `minecraft:honeycomb`

**Пассивные:**
1. **Производство жуков** — каждые 20 секунд создаётся 1 маленький жук-сущность (модель — уменьшенный Silverfish), который следует за игроком. Максимум 10 жуков одновременно. Жуки атакуют ближайших врагов (урон 1❤) и могут подбирать предметы в радиусе 5 блоков (авто-лут в инвентарь игрока).  
2. **Роевое чутьё** — игрок видит контуры всех существ, у которых здоровье ниже 50%, сквозь стены (подсветка красным цветом).  
3. **Королевская феромона** — все членистоногие (пауки, чешуйницы, пещерные пауки) в радиусе 30 блоков не атакуют игрока и не получают от него урон (нейтральны).

**Активные:**
1. **Жертва рою** (кулдаун 30 сек) — клик на моба/игрока: все жуки мгновенно атакуют цель, нанося суммарно 5❤ + 0.5❤ за каждого жука. Цель получает замедление III на 5 секунд.  
   - *Эффект:* жуки светятся красным и издают звук атаки (`entity.player.attack.sweep`).  
   - *Звук:* `entity.zombie.infect` при попадании.  
2. **Улей-крепость** (кулдаун 180 сек) — игрок приседает и за 5 секунд превращается в статичный улей (3×3 блока, модель из блоков медовых сот). Даёт 60% сопротивление урону всем союзникам в радиусе 20 блоков. Улей имеет 30❤ прочности и может быть разрушен врагами.  
   - *Эффект:* во время превращения — частицы `minecraft:spore_blossom_air` и `minecraft:cherry_leaves` жёлтого цвета.  
   - *Звук:* `block.respawn_anchor.charge` и `entity.villager.work_butcher`.  
3. **Улейный щит** (кулдаун 60 сек) — создаёт вокруг игрока барьер из роящихся жуков, который поглощает до 10❤ урона в течение 15 секунд. Барьер визуально представлен вращающимися частицами `minecraft:item_slime` и `minecraft:glow`.  
   - *Эффект:* при получении урона — вспышка `minecraft:flash`.  
   - *Звук:* `entity.vex.charge` при активации и `entity.generic.explode` при разрушении.

**Недостатки:** игрок не может носить шлем (закрывает обзор роя); вода убивает всех жуков и наносит 1❤ урона игроку за каждого погибшего жука; урон от огня увеличен вдвое.

### 6.3. Глубинный Навигатор (Deep Navigator)
**Иконка:** `minecraft:ender_eye`

**Пассивные:**
1. **Межпространственное зрение** — игрок видит координаты ближайшего портала (в Незер, Энд или к другому игроку) в радиусе 200 блоков, даже сквозь стены (подсвечивается фиолетовым).  
2. **Эфемерная плоть** — игрок не получает урон от падения и лавы, но длительность любых зелий (положительных и отрицательных) сокращена вдвое.  
3. **Пространственная память** — игрок автоматически запоминает координаты всех порталов, через которые проходил (отображаются на F3-экране как дополнительные метки).

**Активные:**
1. **Временный разлом** (кулдаун 60 сек) — открывает портал в случайное измерение (Незер/Энд/Обычный мир) на 15 секунд. Все сущности в радиусе 5 блоков засасываются в портал и телепортируются туда.  
   - *Эффект:* портал выглядит как фиолетовый вихрь с частицами `minecraft:portal` и `minecraft:dragon_breath`.  
   - *Звук:* `block.portal.trigger` и `entity.enderman.teleport`.  
2. **Якорь реальности** (кулдаун 240 сек) — создаёт статичный портал, соединяющий точку, где стоит игрок, и точку, куда он смотрит (на расстоянии до 50 блоков). Портал действует 30 секунд и может использоваться любыми сущностями.  
   - *Эффект:* двойная спираль из частиц `minecraft:end_rod` и `minecraft:enchant`.  
   - *Звук:* `block.portal.ambient` и `item.totem.use` при создании.  
3. **Портал в пустоту** (кулдаун 180 сек) — открывает портал в Пустоту (измерение void) на 5 секунд, который засасывает всех в радиусе 10 блоков, нанося 5❤ урона и телепортируя на случайные координаты в мире (с вероятностью 50% — в лаву).  
   - *Эффект:* чёрный вихрь с частицами `minecraft:sculk_charge` и `minecraft:smoke`, сопровождаемый мерцанием.  
   - *Звук:* `entity.wither.spawn` и `entity.ender_dragon.death`.

**Недостатки:** нет естественной регенерации — здоровье восстанавливается только при прохождении через порталы (+1❤ за каждый проход); при отсутствии порталов в радиусе 100 блоков игрок начинает терять 0.5❤ каждые 30 секунд (распад).

### 6.4. Алхимический Монах (Alchemical Monk)
**Иконка:** `minecraft:brewing_stand`

**Пассивные:**
1. **Материальный дисбаланс** — игрок не может использовать верстак, печь, кузницу, точильный камень и другие блоки крафта; все крафты осуществляются через личный «Алхимический круг» (открывается по нажатию специальной клавиши, но это не активка — просто открывает интерфейс).  
2. **Цена творения** — каждый созданный предмет (включая блоки) отнимает 1❤ здоровья. Если здоровье <2❤ — крафт невозможен.  
3. **Круговорот веществ** — при разрушении блока есть 5% шанс получить случайный ресурс вместо обычного выпадающего предмета.

**Активные:**
1. **Золотое касание** (кулдаун 10 сек) — клик на блок (кроме bedrock) превращает его в золотую руду на 30 секунд, после чего блок восстанавливается. Можно добыть золото, но быстро.  
   - *Эффект:* блок покрывается золотыми искрами (`minecraft:spell` и `minecraft:glow`).  
   - *Звук:* `entity.evoker.prepare_wololo`.  
2. **Философский взрыв** (кулдаун 120 сек) — все ингредиенты в инвентаре превращаются в один случайный ценный предмет (алмаз, незерит, тотем) пропорционально количеству, но игрок теряет все остальное.  
   - *Эффект:* взрыв с частицами `minecraft:explosion` и `minecraft:firework` золотого цвета.  
   - *Звук:* `entity.generic.explode` и `item.firecharge.use`.  
3. **Трансмутация плоти** (кулдаун 60 сек) — игрок может превратить 4❤ своего здоровья в 2 алмаза или 1 незеритовый слиток (выбор через интерфейс). При здоровье <6❤ способность не работает.  
   - *Эффект:* вокруг игрока красные частицы `minecraft:dragon_breath`, которые превращаются в алмазные искры.  
   - *Звук:* `entity.witch.throw` и `block.beacon.activate`.

**Недостатки:** игрок не может носить никакую броню (кроме кожанной, которая даёт только 1 защиту); любой урон увеличен на 30%, если в инвентаре более 10 стаков предметов.

### 6.5. Фантомный Археолог (Phantom Archaeologist)
**Иконка:** `minecraft:filled_map`

**Пассивные:**
1. **Чанковое зрение** — игрок видит границы чанков в виде светящихся линий (зелёный контур) и видит, какие чанки загружены, а какие нет.  
2. **Эфемерный инвентарь** — при смерти предметы не выпадают, а сохраняются в «облачном хранилище» (доступно через команду /restore, которую мод добавляет). При этом игрок не может использовать сундуки и шалкеры.  
3. **Бэкап сознания** — при смерти игрок сохраняет 50% накопленного опыта (не теряет всё).

**Активные:**
1. **Кэш-память** (кулдаун 15 сек) — сохраняет текущее состояние инвентаря, здоровья, позиции и эффектов. При повторном использовании восстанавливает сохранённое состояние (как чит-сохранение).  
   - *Эффект:* вокруг игрока появляются частицы `minecraft:enchant_glyph` и `minecraft:glow`, затем всё становится прозрачным на мгновение.  
   - *Звук:* `block.respawn_anchor.charge` при сохранении и `item.totem.use` при восстановлении.

   - *Звук:* `entity.ender_dragon.death` и `block.end_portal.spawn`.

**Недостатки:** игрок получает 0.5❤ урона за каждый звуковой эффект (шаги, дождь, взрывы, даже свои активки). Также не может использовать элитры.

### 6.6. Странник Времени (Time Wanderer)

**Активные:**
1. **Ускорение** (кулдаун 60 сек) — скорость движения, атаки и добычи увеличивается в 3 раза на 10 секунд, но после этого игрок застывает на 5 секунд (замедление V).

   - *Звук:* `entity.illusioner.mirror_move` (быстрое щёлканье).  
2. **Стазис-поле** (кулдаун 120 сек) — все сущности в радиусе 15 блоков замораживаются во времени (не двигаются, не атакуют) на 8 секунд. Игрок также не может двигаться, но может атаковать (урон по замороженным существам удваивается).  
   - *Эффект:* поле из голубых частиц `minecraft:snowflake` (или `minecraft:glow`), всё замедляется.  
   - *Звук:* `entity.elder_guardian.curse` и `block.portal.ambient`.  
3. **Временная петля** (кулдаун 180 сек) — создаёт зону радиусом 15 блоков, в которой все действия (движение, атаки) повторяются через 2 секунды с задержкой (эхо). Длится 10 секунд.  
   - *Эффект:* внутри зоны летают частицы `minecraft:sculk_charge` и `minecraft:sculk_soul`.  
   - *Звук:* циклическое `entity.elder_guardian.curse` и `block.portal.ambient`.

**Недостатки:** игрок не может использовать часы (они взрываются); не видит время суток (всегда темно или светло случайно); урон от падения увеличен вдвое.

### 6.7. Биоморф (Biomorph)
**Иконка:** `minecraft:grass_block`

**Пассивные:**
1. **Адаптация** — в каждом биоме игрок получает уникальные баффы/дебаффы:  
   - Пустыня: +20% скорость, +огнеупорность (иммунитет к огню).

   - Энд: левитация I (постоянно), но +50% урон от падения.  
2. **Метаболизм биома** — игрок может есть только те блоки/предметы, которые растут в текущем биоме (например, в пустыне — кактусы, в лесу — яблоки/грибы, в аду — грибы, в океане — водоросли). Иная еда наносит 1❤ урона и даёт голод.  
3. **Экосистемный импринтинг** — после 5 минут пребывания в биоме игрок получает постоянный +5% к скорости в этом биоме навсегда.

**Активные:**
1. **Смена биома** (кулдаун 180 сек) — на 30 секунд игрок «носит» выбранный биом с собой (можно выбрать из списка доступных биомов) и получает его баффы везде.  
   - *Эффект:* вокруг игрока появляются частицы, характерные для выбранного биома (например, в пустыне — `minecraft:sand`, в лесу — `minecraft:cherry_leaves`).  
   - *Звук:* `entity.evoker.prepare_wololo` и `block.beacon.activate`.  
2. **Биомный взрыв** (кулдаун 60 сек) — выпускает волну, превращающую блоки в радиусе 10 блоков в блоки текущего биома (например, вода → лёд, земля → песок). Враги получают 3❤ урона от смены блоков.  
   - *Эффект:* взрывная волна из частиц соответствующего биома (песок, листья, снежинки).  
   - *Звук:* `entity.generic.explode` и `block.lava.pop`.  
3. **Биомный захват** (кулдаун 120 сек) — временно (на 1 минуту) превращает весь биом в радиусе 50 блоков в выбранный биом. Все растения, мобы и структуры меняются соответственно. После окончания эффекта всё возвращается.  
   - *Эффект:* плавная смена травы, воды, деревьев с частицами `minecraft:spore_blossom_air`.  
   - *Звук:* `entity.illusioner.prepare_mirror` и `block.end_portal.spawn`.

**Недостатки:** при переходе между биомами игрок получает 1❤ урона (шок); не может строить из блоков другого биома (они разрушаются через 5 секунд).

**Недостатки:** игрок не может атаковать сам (урон равен 0); если все заражённые существа погибают, игрок получает 5❤ урона (синдром отмены); игрок не может использовать зелья и тотемы.

   - *Эффект:* игрок покрывается кристаллами (частицы `minecraft:spell` и `minecraft:glow`).  
   - *Звук:* `item.armor.equip_diamond` и `block.respawn_anchor.charge`.  
3. **Землетрясение** (кулдаун 180 сек) — наносит 5❤ урона всем существам в радиусе 20 блоков и создаёт трещины в земле (блоки превращаются в гравий).  
   - *Эффект:* масштабное землетрясение с частицами `minecraft:campfire_cosy_smoke` и `minecraft:explosion`.  
   - *Звук:* `entity.generic.explode` (громко) и `entity.warden.sonic_boom`.

**Недостатки:** игрок тонет в любой жидкости (вода, лава) мгновенно (как асфальт); не может использовать лодки, элитры, зелья скорости.

### 6.14. Радиоактивный Фантом (Radioactive Phantom)
**Иконка:** `minecraft:experience_bottle`

**Пассивные:**
1. **Постоянное облучение** — игрок постоянно наносит 0.5❤ урона всем существам (включая мобов и игроков) в радиусе 5 блоков каждые 5 секунд. Сам игрок иммунен к этому.  
2. **Гниение плоти** — игрок не может есть; голод восполняется от смерти мобов (каждый убитый моб даёт +2 сытости).  
3. **Счётчик Гейгера** — игрок визуально видит уровень радиации вокруг (чем ближе к нему, тем ярче свечение частиц `minecraft:glow`).

**Активные:**
1. **Волна распада** (кулдаун 60 сек) — выпускает конусное излучение на 20 блоков, нанося 6❤ урона всем и давая эффект «увядание II» на 10 секунд.  
   - *Эффект:* зелёная волна с частицами `minecraft:witch` и `minecraft:spore_blossom_air`.  
   - *Звук:* `entity.witch.throw` и `entity.evoker.cast_spell`.  
2. **Радиоактивный скачок** (кулдаун 15 сек) — телепортируется на 10 блоков вперёд, оставляя облако, которое наносит 2❤ урона в секунду всем, кто войдёт (облако живёт 5 секунд).  
   - *Эффект:* облако из частиц `minecraft:cloud` и `minecraft:smoke`, зелёного цвета.  
   - *Звук:* `entity.enderman.teleport` и `block.fire.extinguish`.  
3. **Мутагенный взрыв** (кулдаун 120 сек) — накладывает случайные эффекты (скорость, сила, слабость, слепота, регенерация, увядание) на всех существ в радиусе 15 блоков на 15 секунд. Эффекты выбираются случайно для каждой цели.  
   - *Эффект:* разноцветная вспышка с частицами `minecraft:effect` и `minecraft:enchant`.  
   - *Звук:* `entity.lightning_bolt.thunder` и `entity.evoker.prepare_attack`.

**Недостатки:** все деревянные, бумажные, шерстяные блоки в радиусе 10 блоков от игрока постепенно превращаются в уголь (через 1 минуту); дождь наносит 0.5❤ урона каждый тик.

### 6.15. Судья Измерений (Dimension Judge)
**Иконка:** `minecraft:end_portal_frame`

**Пассивные:**
1. **Закономерность** — урон игрока по существам, которые его не атаковали, равен 0 (не может ударить первым). Но если игрока ударили, его следующий удар (в течение 5 секунд) наносит 5❤ и игнорирует броню.  
2. **Всевидящее око** — игрок видит уровень здоровья, брони, эффекты и количество предметов у всех игроков в радиусе 50 блоков (в виде иконок над головами).  
3. **Баланс сил** — если у игрока и у врага равное количество ❤, битва заканчивается ничьей (оба получают 1❤ урона, и атака отменяется).

**Активные:**
1. **Приговор** (кулдаун 30 сек) — клик на существо: если у цели больше здоровья, чем у игрока, она получает урон, равный разнице; если у игрока больше — игрок делится здоровьем с целью (лечит её).  
   - *Эффект:* световой луч между игроком и целью, частицы `minecraft:flash` и `minecraft:glow`.  
   - *Звук:* `entity.evoker.prepare_wololo` и `block.beacon.activate`.  
2. **Аннигиляция** (кулдаун 300 сек) — игрок выбирает игрока или моба, через 10 секунд (огромная визуальная анимация) цель мгновенно уничтожается (смерть без возрождения), но игрок теряет всё здоровье, оставляя 0.5❤.  
   - *Эффект:* небо становится чёрным, цель окутывается частицами `minecraft:sculk_charge` и `minecraft:dragon_breath`, затем взрыв.  
   - *Звук:* `entity.ender_dragon.death` и `entity.wither.spawn`.  
3. **Балансировка** (кулдаун 180 сек) — уравнивает здоровье всех игроков в радиусе 30 блоков до среднего значения (сумма всех ❤, делённая на количество игроков).  
   - *Эффект:* волна золотых частиц `minecraft:enchant_glyph` и `minecraft:glow`.  
   - *Звук:* `block.beacon.power_select` и `entity.evoker.prepare_summon`.

**Недостатки:** игрок не может использовать тотемы бессмертия и зелья регенерации; не может строить алтари/храмы (они для него запрещены — при попытке разместить блок алтаря он взрывается).

## 7. PvE-РАСЫ (3 ШТУКИ)

### 7.1. Мицелиальный Симбионт (Mycelial Symbiont)
**Иконка:** `minecraft:brown_mushroom`

**Пассивные:**
1. **Мицелиальная сеть** — все блоки, размещённые игроком, постепенно покрываются грибницей (визуально). Игрок «чувствует» все блоки в радиусе 30 блоков от любой размещённой им грибницы (видит их содержимое, руду, мобов).  
2. **Питание спорами** — игрок не ест обычную еду; голод восполняется костной мукой (+3 сытости) и гнилой плотью (+1 сытость). При этом вокруг игрока появляется зона спор (как ковёр), которая замедляет врагов на 20%, но и игрока тоже.  
3. **Грибное зрение** — игрок видит все грибы (любого типа) в радиусе 50 блоков сквозь стены.

**Активные:**
1. **Спороносный урожай** (кулдаун 30 сек) — клик на любой твёрдый блок (земля, камень, дерево) превращает его в случайный ресурс (уголь, железо, золото, редстоун, алмаз) в зависимости от глубины (на Y=12 шанс алмаза 40%). При неудаче выпадает гниль.  
   - *Эффект:* из блока вырываются споры (частицы `minecraft:spore_blossom_air` и `minecraft:cherry_leaves`).  
   - *Звук:* `item.bone_meal.use` и `block.grass.break`.  
2. **Грибной побег** (кулдаун 90 сек) — создаёт «живой портал» из мицелия на расстоянии до 50 блоков, куда можно телепортироваться один раз (и обратно тоже). Портал живёт 20 секунд и требует костной муки для активации.  
   - *Эффект:* портал из грибов с частицами `minecraft:portal` и `minecraft:spore_blossom_air`.  
   - *Звук:* `block.portal.trigger` и `entity.enderman.teleport`.  
3. **Споровый туман** (кулдаун 60 сек) — создаёт облако спор в радиусе 10 блоков, которое даёт регенерацию I союзникам и наносит 1❤ урона/сек врагам. Длится 15 секунд.  
   - *Эффект:* облако из частиц `minecraft:cloud` и `minecraft:spore_blossom_air` зелёного цвета.  
   - *Звук:* `entity.witch.throw` и `block.beacon.activate`.

**Недостатки:** игрок не может удаляться от своей мицелиальной сети более чем на 20 блоков — иначе начинает терять 0.5❤/сек (отрыв от грибницы). Вода и дождь смывают мицелий и наносят урон. Не может использовать печи/верстаки — только грибные аналоги.

### 7.2. Глубинный Геомант (Deep Geomancer)
**Иконка:** `minecraft:diamond_pickaxe` (но с изменённой текстурой)

**Пассивные:**
1. **Рудное зрение** — игрок видит контуры всех руд (включая скрытые) в радиусе 15 блоков сквозь стены, но только если смотрит на них не менее 2 секунд (фокусируется).  
2. **Каменная плоть** — игрок регенерирует здоровье, стоя на каменных или земляных блоках (+1❤ за 10 секунд, но только если у него >=4❤).  
3. **Земляной слух** — игрок слышит мобов в радиусе 20 блоков под землёй (звук шагов усиливается).

**Активные:**
1. **Вытягивание руды** (кулдаун 10 сек) — клик по блоку руды (железо, золото, алмаз, уголь и т.д.) на расстоянии до 10 блоков: руда исчезает, в инвентарь падает 1-2 единицы ресурса, но игрок теряет 1❤ здоровья за каждый кусок (если здоровья <2❤ — способность не работает).  
   - *Эффект:* руда светится и исчезает с частицами `minecraft:smoke` и `minecraft:spell`.  
   - *Звук:* `block.stone.break` и `entity.experience_orb.pickup`.  
2. **Сдвиг породы** (кулдаун 60 сек) — создаёт туннель длиной до 15 блоков в любом направлении (каменные блоки исчезают, оставляя пустое пространство). Можно также запечатать проход (восстановить блоки камня). Каждый изменённый блок отнимает 0.5❤.  
   - *Эффект:* земля расступается с частицами `minecraft:explosion` и `minecraft:smoke`.  
   - *Звук:* `entity.generic.explode` и `block.grass.break`.  
3. **Каменный кулак** (кулдаун 30 сек) — игрок наносит мощный удар в направлении взгляда, создавая ударную волну, которая наносит 4❤ урона и отбрасывает врагов на 5 блоков.  
   - *Эффект:* волна из частиц `minecraft:sonic_boom` и `minecraft:flame`.  
   - *Звук:* `entity.warden.sonic_boom` и `entity.player.attack.sweep`.

**Недостатки:** игрок не может использовать обычные кирки (ломаются моментально) — только незеритовая, но с ней добыча в 3 раза медленнее. Не может носить броню (кроме кожанной) — металл притягивает рудные жилы и наносит урон. На высоте >Y80 теряет 1❤ каждые 30 секунд.

### 7.3. Странствующий Садовник (Wandering Gardener)
**Иконка:** `minecraft:oak_sapling`

**Пассивные:**
1. **Зелёная кровь** — игрок регенерирует здоровье только когда стоит на траве или листве (+1❤/5 сек). На камне или в пустыне регенерация отсутствует, голод падает вдвое быстрее.  
2. **Мирная душа** — прямой урон игрока (руками, мечом, луком) равен 0. Вместо этого игрок может **приручать** мобов, давая им семена, яблоки, морковь — они становятся союзниками (до 5 существ одновременно).  
3. **Язык цветов** — игрок понимает, когда растениям нужна вода или свет (отображается в виде сообщений в чат или на HUD).

**Активные:**
1. **Благословение роста** (кулдаун 30 сек) — клик на землю/траву: в радиусе 5 блоков мгновенно вырастает случайный набор растений (пшеница, морковь, картофель, свекла, цветы, деревья). Растения живут 2 минуты, затем увядают (если не собрать).  
   - *Эффект:* частицы `minecraft:spore_blossom_air` и `minecraft:cherry_leaves`, зелёный свет.  
   - *Звук:* `item.bone_meal.use` и `block.grass.place`.  
2. **Призыв дождя** (кулдаун 180 сек) — вызывает дождь в радиусе 50 блоков на 20 секунд, ускоряя рост растений в 10 раз. Сам игрок в дожде получает слабость и медлительность.  
   - *Эффект:* дождевые частицы `minecraft:rain` и `minecraft:splash`.  
   - *Звук:* `entity.lightning_bolt.thunder` и `ambient.weather.rain`.  
3. **Призыв дриады** (кулдаун 120 сек) — призывает древесного голема (аналог железного, но из дерева, 30❤, урон 5❤), который атакует врагов игрока. Живёт 1 минуту.  
   - *Эффект:* из земли вырастает дерево, из которого появляется голем с частицами `minecraft:spore_blossom_air`.  
   - *Звук:* `entity.iron_golem.repair` и `block.wood.break`.

**Недостатки:** игрок не может добывать дерево (рубить) — это наносит 3❤ урона. Не может носить оружие (мечи, топоры, луки) — они превращаются в цветы. Привязан к зелёным биомам — в пустыне, аду, горах теряет здоровье (0.5❤/10 сек).

## 8. КОНФИГУРАЦИЯ (CLOTH CONFIG)

- Создать экран конфигурации с вкладками по каждой расе.
- В каждой вкладке — все числовые параметры с ползунками или полями ввода.
- Значения по умолчанию — как указано в ТЗ.
- Параметры: кулдауны (сек), урон (❤), длительности (сек), шансы (%), радиусы (блоки), стоимость опыта/здоровья и т.д.
- После изменения конфигурации перезагрузка мира не требуется (параметры применяются на лету).

## 9. КОМПОНЕНТЫ CARDINAL COMPONENTS

Создать следующие компоненты для хранения данных игроков:

- `swarm_data`: целое число (количество жуков) и список UUID жуков.
- `parasite_data`: список UUID заражённых существ.
- `mutation_data`: счётчик убитых мобов.
- `time_data`: сохранённое состояние для «Кэш-памяти» (инвентарь, здоровье, позиция).
- `biome_data`: сохранённый выбранный биом для «Смены биома».
- `lunar_data`: текущая фаза луны и время до смены.
- `mycelium_data`: список координат размещённой грибницы.
- `gardener_data`: список UUID прирученных мобов.

Компоненты должны сохраняться при выходе из мира и загружаться при входе.

## 11. ПРИМЕРЫ JSON (ДЛЯ ОРИЕНТИРА)

Ниже приведён пример для одной пассивки и одной активки, чтобы разработчик понимал формат.

**Пассивка (атрибут):**
```json
{
  "type": "origins:attribute_modifier",
  "modifier": {
    "attribute": "minecraft:generic.movement_speed",
    "operation": "multiply_total",
    "value": -0.5
  },
  "name": "Медлительность",
  "description": "Ты медленнее на 50%"
}
```

**Активка (с эффектом и кулдауном):**
```json
{
  "type": "origins:active_self",
  "entity_action": {
    "type": "origins:apply_effect",
    "effect": {
      "effect": "minecraft:speed",
      "duration": 200,
      "amplifier": 1
    }
  },
  "cooldown": 60,
  "hud_render": {
    "should_render": true,
    "sprite_location": "origins:textures/gui/resource_bar.png",
    "bar_index": 0
  },
  "key": {
    "key": "key.origins.primary_active"
  },
  "name": "Ускорение",
  "description": "На 10 сек ты получаешь скорость II"
}
```
